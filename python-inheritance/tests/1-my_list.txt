# 1-my_list.txt

=======================
How to Use 1-my_list.py
=======================

This module defines a class ``MyList`` that inherits from ``list``.

Instantiation
=============

::

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> print(my_list)
    []

Usage
=====

Elements can be added like a normal list.

::

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

print_sorted Method
===================

The method prints the list in ascending sorted order.

::

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

The original list is not modified.

::

    >>> print(my_list)
    [1, 4, 2, 3, 5]

Works with empty lists.

::

    >>> empty = MyList()
    >>> empty.print_sorted()
    []

Sorting also works with negative numbers.

::

    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(-1)
    >>> my_list.append(2)
    >>> print(my_list)
    [3, -1, 2]

::

    >>> my_list.print_sorted()
    [-1, 2, 3]

::

    >>> print(my_list)
    [3, -1, 2]
